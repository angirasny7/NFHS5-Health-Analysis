# 🩺 NFHS-5 Health Indicators Analysis (India)

This project analyzes major health indicators across Indian states using NFHS-5 (2019–21) data.

It includes data cleaning, statistical analysis, visualization, and an interactive map.

---

## 📊 Objectives

- Analyze obesity, anaemia, hypertension, and blood sugar across states
- Study correlations between health indicators
- Compare gender differences in health conditions
- Visualize regional patterns
- Build an interactive health dashboard

---

## 📁 Dataset

NFHS-5 State-Level Dataset (2019–21)

Source: https://github.com/pratapvardhan/NFHS-5

---

## 📈 Key Visualizations

- Correlation heatmap
- Hypertension comparison (men vs women)
- Anaemia vs obesity scatter plot
- Interactive India choropleth map
